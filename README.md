# msc (WIP)

## Prerequisites

- conda >= 23.5.0
- git

## Setup

Run this in your terminal (for UNIX-based systems):

```sh
git clone https://github.com/f1lem0n/msc.git
cd msc
conda env create -f envs/msc.yaml -y
```

## Usage

1. Firstly `conda activate msc`.
1. Run `python3 mccaskill.py <RNA_sequence>`
